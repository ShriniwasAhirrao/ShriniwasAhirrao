## About

I build data tools that make messy, real-world data actually usable.

- Published **[ParseIQ](https://github.com/ShriniwasAhirrao/ParseIQ)** on PyPI — an AI-powered data quality agent used by engineers to profile and score any data file in one command
- Competed at **IIT Techfest 2023** (Godrej & Boyce challenge) — built a drone system that reads QR codes and streams data to a live dashboard
- Work across the full data stack: ingestion, profiling, ML modelling, and visualisation (Tableau)
